# Internationalization (i18n) Guide

## Overview

Chrono-Walk supports 11 languages with proper timezone handling and localized number/date formatting.

## Setup

### Installation

```bash
npm install i18next react-i18next i18next-browser-languagedetector date-fns-tz
```

### Configuration

```javascript
// src/i18n/config.js
import i18n from 'i18next';
import LanguageDetector from 'i18next-browser-languagedetector';
import { initReactI18next } from 'react-i18next';
import enTranslations from './locales/en.json';
// ... import other languages

i18n
  .use(LanguageDetector)
  .use(initReactI18next)
  .init({
    resources: {
      en: { translation: enTranslations },
      es: { translation: esTranslations },
      // ... all languages
    },
    fallbackLng: 'en',
    interpolation: { escapeValue: false }
  });
```

### Initialize in App

```javascript
// src/main.jsx
import './i18n/config.js';  // Import to initialize

import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
```

## Usage in Components

### Basic Translation

```javascript
import { useTranslation } from 'react-i18next';

export function Header() {
  const { t } = useTranslation();
  
  return (
    <header>
      <h1>{t('title')}</h1>
      <p>{t('subtitle')}</p>
    </header>
  );
}
```

### With Variables (Interpolation)

```json
// locales/en.json
{
  "welcome": "Welcome, {{name}}!",
  "computationTime": "Computation took {{time}} seconds"
}
```

```javascript
const { t } = useTranslation();

return (
  <div>
    <p>{t('welcome', { name: 'John' })}</p>
    <p>{t('computationTime', { time: 5.2 })}</p>
  </div>
);
```

### Pluralization

```json
{
  "simulationCount": "You have {{count}} simulation",
  "simulationCount_plural": "You have {{count}} simulations"
}
```

```javascript
const { t } = useTranslation();

return <p>{t('simulationCount', { count: results.length })}</p>;
```

### Language Switching

```javascript
import { useTranslation } from 'react-i18next';
import { SUPPORTED_LANGUAGES, changeLanguage } from '../i18n/config';

export function LanguageSwitcher() {
  const { i18n } = useTranslation();
  
  return (
    <div className="language-switcher">
      {SUPPORTED_LANGUAGES.map(lang => (
        <button
          key={lang.code}
          onClick={() => changeLanguage(lang.code)}
          active={i18n.language === lang.code}
        >
          {lang.flag} {lang.name}
        </button>
      ))}
    </div>
  );
}
```

## Date/Time Formatting

### Locale-Aware Formatting

```javascript
import { 
  formatLocalDateTime,
  formatLocalDate,
  formatLocalTime,
  getUserTimezone 
} from '../i18n/dateTimeUtils';

export function ResultsTimestamp({ timestamp, language }) {
  const timezone = getUserTimezone(); // Auto-detect
  
  return (
    <div>
      <p>{formatLocalDateTime(timestamp, language, timezone)}</p>
      <p>{formatLocalDate(timestamp, language)}</p>
      <p>{formatLocalTime(timestamp, language)}</p>
    </div>
  );
}
```

### Relative Time

```javascript
import { formatRelativeTime } from '../i18n/dateTimeUtils';

export function LastRun({ lastRunTime, language }) {
  return <p>Last run: {formatRelativeTime(lastRunTime, language)}</p>;
}
```

### Computation Time Formatting

```javascript
import { formatComputationTime } from '../i18n/dateTimeUtils';

export function ComputationStats({ timeMs }) {
  return <p>Took: {formatComputationTime(timeMs)}</p>;
}
```

## Number & Currency Formatting

```javascript
import { formatNumber, formatCurrency } from '../i18n/dateTimeUtils';

export function Statistics({ mean, cost, language }) {
  return (
    <div>
      <p>Mean: {formatNumber(mean, language, { 
        maximumFractionDigits: 4 
      })}</p>
      <p>Cost: {formatCurrency(cost, 'USD', language)}</p>
    </div>
  );
}
```

## Adding New Language

### 1. Create Translation File

```json
// src/i18n/locales/la.json
{
  "title": "Chrono-Walk",
  "subtitle": "Simulator Processum Stochasticum",
  "modes": { ... }
}
```

### 2. Update Config

```javascript
// src/i18n/config.js
import laTranslations from './locales/la.json';

i18n.init({
  resources: {
    // ...
    la: { translation: laTranslations },
  }
});

export const SUPPORTED_LANGUAGES = [
  // ...
  { code: 'la', name: 'Latin', nativeName: 'Latīnus', flag: '🏛️' },
];
```

### 3. Update Date Formatter (if needed)

```javascript
// src/i18n/dateTimeUtils.js
import { la } from 'date-fns/locale';

const localeMap = {
  // ...
  'la': la,
};
```

## Translation Best Practices

### ✅ Do's

- Keep keys hierarchical: `modes.cycleAnalysis`
- Use context-specific keys: `button.run` not just `run`
- Provide translator notes in your code
- Keep text as short as possible
- Avoid hardcoding dates/times/numbers

### ❌ Don'ts

- Don't translate everything in one string
- Don't use `t()` in loops without key
- Don't forget interpolation for variables
- Don't forget plural forms
- Don't assume English word order in other languages

## Timezone Handling

### Get User Timezone

```javascript
const userTimezone = Intl.DateTimeFormat().resolvedOptions().timeZone;
// Returns: "America/New_York", "Europe/London", etc.
```

### Convert UTC to User Timezone

```javascript
import { utcToZonedTime, format } from 'date-fns-tz';

const utcDate = new Date('2024-04-01T12:00:00Z');
const timezone = 'Asia/Tokyo';
const zonedDate = utcToZonedTime(utcDate, timezone);

console.log(format(zonedDate, 'PPpp')); // Local format
```

## Performance Optimization

### Lazy Load Translations

```javascript
i18n.use(HttpApi).init({
  backend: {
    loadPath: '/locales/{{lng}}.json'
  },
  detection: { order: ['localStorage', 'navigator'] }
});
```

### Bundle Only Used Languages

```javascript
// webpack.config.js
new BundleAnalyzerPlugin({
  analyzerMode: 'static',
  exclude: ['unused-locales/**']
});
```

## Testing i18n

```javascript
// __tests__/i18n.test.js
import i18n from '../i18n/config';

describe('i18n', () => {
  it('should translate keys', () => {
    expect(i18n.t('title')).toBe('Chrono-Walk');
  });
  
  it('should interpolate variables', () => {
    const result = i18n.t('welcome', { name: 'John' });
    expect(result).toBe('Welcome, John!');
  });
  
  it('should support all languages', () => {
    SUPPORTED_LANGUAGES.forEach(lang => {
      i18n.changeLanguageAsync(lang.code);
      expect(i18n.language).toBe(lang.code);
    });
  });
});
```
